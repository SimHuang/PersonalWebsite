PROJECTS

Ango Web Application

Ango is a web application made to allow students to talk about their schools, whether it's a interesting story or some questions they have about their school. I use Node JS with express JS and MySQL for my backend technologies. The frontend consists of HTML5, CSS3, and Angular JS. This is a project for me to learn about these new javascript technologies and it is still work in progress. The source code can be found in my github.

Quench Your Thirst Dating Web Application

This web application is a group based project including two other team members and me. It is a fully functional dating website which allow users to create a new account, browse other user’s profiles, request/cancel dates with other users etc. We designed our database using the E-R relational model. We used HTML5, CSS3, Javascript for the user interface. Backend technologies such as Java, JSP, JDBC, and SQL was used for interaction with our MySQL database.

PathX Desktop Game Application

This is a desktop game application. It is written in Java using the Java swing for the GUI and XML for resource storage. Since this was a fairly large project, I designed all the classes using violet UML before implementing them. I used subversion version control for project maintenance. This application utitlizing Java multithreading to allow user to interact with moving enemy pieces. It contains multiple screens for user interaction and multiple levels of gameplay.
